html tags

Always have an opening tag and closing tag and the content is between these tags
example:  
```
<head> 
    content 
</head>
``` 

some tags don't have a closing tag 
examples:

```
<!--this is line break-->
<br>
<!--this is horizontal line-->
<hr>  
```
html tags have attributes
example:

```
<p id="myID">
</p>
```
